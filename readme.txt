Provides endpoint and integration with Broadbean's AdCourier API, as
described here:

http://api.adcourier.com/docs/index.cgi?page=jobboards_overview

== Description ==

This plugin adds an endpoint for receiving Broadbean AdCourier job postings.

It also adds a new 'Job' custom post type.

When job postings are received from Broadbean, Job posts are generated with
all available metadata.

== Installation ==

1. Upload wp-broadbean/ directory to the wp-content/plugins directory.
2. Activate Broadbean through the 'Plugins' menu in Wordpress.

== Frequently Asked Questions ==

= Who is this for? =

Anyone who uses Broadbean AdCourier to manage ad postings, and would like
those postings to automatically appear on their own Wordpress site.
